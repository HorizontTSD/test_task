# Запуск на своей машине

#### Установка зависимостей
```bash
pip install pdm
pdm install
```


Активация окружения
```bash
source .venv/bin/activate
```

Добавление новых зависимостей
```bash
pdm add pandas
```
Удаление  зависимостей
```bash
pdm remove pandas
```
